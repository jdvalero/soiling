General Information

DBDesigner 4 is a free available database design system that integrates database design, modelling, creation and maintenance into a single, seamless environment.

It is written for Microsoft© Windows 2k/XP© and Linux KDE/Gnome to support both, the Windows and the Linux platform.

DBDesigner 4 is developed and optimized for the MySQL-Database.
